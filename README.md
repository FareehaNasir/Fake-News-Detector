## 📖 About The Project

This is a **Fake News Detection System** that uses Artificial Intelligence to detect misinformation. The system analyzes both **text** and **images** together to decide if content is real or fake.

### What it does:

| Feature | Description |
|---------|-------------|
| Text Analysis | Checks news articles for fake patterns |
| Image Analysis | Detects manipulated or edited images |
| Multi-Modal Fusion | Combines text + image results |
| Credibility Score | Gives 0-100% score |
| Verdict | Says REAL / SUSPICIOUS / FAKE |
| Audit Ledger | Saves all checks in database |
